# 💊 MediFinder

MediFinder is a MERN Stack web application that helps users find nearby pharmacies with the required medicines in stock. Pharmacies can manage their inventory, while users can search medicines, reserve them, and locate the nearest pharmacy.

## 🚀 Features

### 👤 User
- User Registration & Login (JWT Authentication)
- Manage Profile
- Add Multiple Addresses
- Select Location using Google Maps
- Search Medicines
- View Nearby Pharmacies
- Reserve Medicines
- View Order History

### 🏥 Pharmacy
- Pharmacy Registration
- Business Profile Management
- Add/Edit/Delete Medicines
- Manage Inventory
- Receive Reservation Requests
- Update Order Status

## 🛠 Tech Stack

### Frontend
- React.js
- React Router
- Tailwind CSS
- Axios
- Google Maps API

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- Multer
- Cloudinary
